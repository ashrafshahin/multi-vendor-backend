=== solution === await user.save() = means save everything... BUT

শুধু token add করছো। name, email, password change করছো না।

তাই: await user.save({ validateBeforeSave:false });

 √ this will give you:
refresh token save হবে ✅
password regex validation আর চলবে না ✅
login error যাবে ✅
bcrypt change করতে হবে না ✅
